# Понятие множества

**Множество** это совокупность различных объектов, собранных по общему признаку, рассматриваемых как единое целое. Например множество книг на полке, множество футболок в шкафу. Объекты из которых состоят множества называют их элементами.  Принадлежность элемента $a$ множеству $P$ записываю так: $a \in P$, где $\in$ знак принадлежности.  Если элемент не принадлежит множеству $P$ то пишут так: $a \notin P$  

Множество может содержать любое количество элементов, конечное и бесконечное. Множество не содержащее ни одного элемента называется **пустым** множеством и обозначается так: Ø. Множество содержащее один элемент называется синглетоном.

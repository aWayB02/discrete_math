
# Множества. 1.1

**Множество** $-$ это совокупность определённых различных объектов, рассматриваемых как единое целое. Например множество книг на полке, множество футболок в шкафу. Объекты из которых состоят множества называют их элементами.  Принадлежность элемента $a$ множеству $P$ записываю так: $a \in P$, где $\in$ знак принадлежности.  Если элемент не принадлежит множеству $P$ ,то пишут так: $a \notin P$  

Множество может содержать любое количество элементов, конечное и бесконечное. Множество не содержащее ни одного элемента называется **пустым** множеством и обозначается так: Ø. Множество содержащее один элемент называется синглетоном.  
  
  
# Способы задания множества

1. Путем прямого перечисления его элементов. Например: $A = \{1, 2, 3\}$, говорит о том, что множество $A$ состоит из $3$ элементов: $1, 2, 3$.
2. При помощи специального условия, с помощью которого любой элемент который удовлетворяет свойству, входит в множество: $A = \{x \ | \ P(x)\}$

Например множество чисел от $1$ до $10$ можно задать следующим образом:  
$P = \{x \ | 1 <= x <= 10 \land x - целое \}$, где знак $\land$ используется как логическое **И**, допускается и такая запись:  $P = \{x \ | 1 <= x <= 10; \ x - целое \}$.   


### Равные множества

Множества называются равными, если они состоят из одинаковых элементов, при этом порядок элементов не имеет значения. Например: $\{a, b, c, d\} = \{b, c, a, d\}$. Равными множествами также могут быть множества, заданные перечислительным путем. Например: 
$\{x \ | \ 0 < x <= 10; x - четное \ число\} = \{2, 4, 6, 8, 10\}$. Стоит отметить, что обычно множество обозначают заглавными латинскими буквами.  


**Кардинальное число** $-$ это число, показывающие сколько элементов содержит множество. Для его обозначения часто используют две вертикальные черты, например пусть имеется множество: $A = \{1, 2, 3, 4\}$, тогда $|A| = 4$. Множество с одинаковым кардинальным число называется эквивалентными. Замечание: каждый элемент множества входит в него ровно по одному разу, то есть элементы множества уникальны.  

Например: $P = \{1, 1, 2\} = \{1, 2\}, \ |P| = 2$.

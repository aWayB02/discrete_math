# 1.4. Объединение множеств

## Определение

**Объединением** (суммой) множеств $A_1, A_2, \ldots, A_n$ называется множество, состоящее из всех элементов, которые принадлежат хотя бы одному из этих множеств.

Формально:
$A = A_1 \cup A_2 \cup \cdots \cup A_n = \lbrace x \mid x \in A_1 \lor x \in A_2 \lor \cdots \lor x \in A_n \rbrace$

где символ $\lor$ обозначает логическую операцию **ИЛИ** (дизъюнкцию).

## Пример

Пусть даны множества:
$A = \\{1, 2, 3\\}, \quad B = \\{a, b, c\\}.$

Их объединение:
$A \cup B = \\{1, 2, 3, a, b, c\\}.$

На диаграмме Эйлера-Венна объединение соответствует области, покрываемой всеми рассматриваемыми множествами.

## Свойства операции объединения

### 1. Коммутативность (перестановочность)
$A \cup B = B \cup A.$

Это свойство справедливо и для любого конечного числа множеств:
$A_1 \cup A_2 \cup \cdots \cup A_n = A_{\sigma(1)} \cup A_{\sigma(2)} \cup \cdots \cup A_{\sigma(n)},$
где $\sigma$ — произвольная перестановка индексов $1, 2, \ldots, n$.

### 2. Ассоциативность (сочетательность)
$$
(A \cup B) \cup C = A \cup (B \cup C).
$$

Благодаря этому свойству можно не указывать порядок выполнения операций и писать просто $A \cup B \cup C$.

### 3. Свойства, связанные с отношением включения

Если $A \subseteq B$, то $A \cup B = B$.

В частности, если $A = B$, то $A \cup B = A = B$.

### 4. Свойства, связанные с особыми множествами

- **Идемпотентность:** $A \cup A = A$.
- **Объединение с пустым множеством:** $A \cup \varnothing = A$.
- **Объединение с универсальным множеством:** $A \cup U = U$ (где $U$ — универсальное множество).

Эти свойства следуют непосредственно из определения объединения и могут быть доказаны с помощью логических преобразований.
